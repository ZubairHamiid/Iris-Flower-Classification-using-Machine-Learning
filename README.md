# 🌸 Iris Flower Classification using Machine Learning

A complete **Machine Learning Classification** project that predicts the species of an Iris flower using the **K-Nearest Neighbors (KNN)** algorithm.

This project demonstrates an end-to-end supervised machine learning workflow—from data exploration and visualization to model training and evaluation—using the famous **Iris Dataset**.

---

## 📖 Project Overview

The objective of this project is to classify iris flowers into one of the following three species:

- 🌼 **Setosa**
- 🌺 **Versicolor**
- 🌷 **Virginica**

The prediction is based on four botanical measurements:

- 📏 Sepal Length
- 📏 Sepal Width
- 🌿 Petal Length
- 🌿 Petal Width

This project follows industry-standard data science practices and serves as an excellent beginner-friendly machine learning project.

---

## 🎯 Project Objectives

- 📂 Load and understand the Iris dataset
- 🔍 Perform Exploratory Data Analysis (EDA)
- 📊 Visualize feature distributions and relationships
- 🧹 Prepare the dataset for machine learning
- 🤖 Train a K-Nearest Neighbors (KNN) model
- 📈 Evaluate model performance
- ✅ Predict iris flower species accurately

## 📂 Dataset

This project uses the famous **Iris Dataset**, one of the most popular datasets in machine learning.

The dataset contains:

- ✅ 150 Flower Samples
- ✅ 4 Numerical Features
- ✅ 3 Flower Species

### Features

| Feature | Description |
|----------|-------------|
| Sepal Length | Length of the sepal (cm) |
| Sepal Width | Width of the sepal (cm) |
| Petal Length | Length of the petal (cm) |
| Petal Width | Width of the petal (cm) |

### Target Classes

- 🌼 Setosa
- 🌺 Versicolor
- 🌷 Virginica

## 🤖 Machine Learning Model

### Algorithm Used

✅ **K-Nearest Neighbors (KNN)**

KNN is a simple yet powerful supervised learning algorithm that classifies new observations based on the majority class of their nearest neighbors.
